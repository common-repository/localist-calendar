== Installation ==

Install the Localist Calendar for WordPress plugin via the WordPress plugin directory, or by uploading the files manually to your server. After becoming a Localist customer, use the plugin to connect to your calendar and enable event listing features.

== Usage ==

= How do I use the plugin? =

After installing the **Localist Calendar for WordPress** plugin, you can begin setting up new widgets in the Localist Calendar section of your WordPress administration.

To begin, mouseover _Localist Calendar_ in the navigation bar and click _Add New Widget_. Paste in your _Localist Calendar's URL_ and click _Configure_. Define your criteria, choose a name for the widget, then click _Save as WordPress Widget_. Once the widget is added, you can add it to a page, post, or theme in the appropriate areas of WordPress.
